#  Project Objective

This repository aims to:

- Build strong foundations in NumPy for data manipulation  
- Provide practical examples for each NumPy concept  
- Serve as a quick reference guide for NumPy operations  
- Prepare learners for advanced Data Science and ML libraries (Pandas, Scikit-learn, TensorFlow)  
- Enable hands-on practice through interactive Jupyter notebooks

#  Topics Covered

| Module | Topic | Concepts Covered |
|--------|-------|------------------|
| 01. | **NumPy Arrays** | ndarray introduction, Array vs lists, dtype, ndim, shape, Memory efficiency |
| 02. | **Array Creation Methods** | np.array(), zeros(), ones(), arange(), linspace(), eye(), identity(), empty(), full() |
| 03. | **Shape, Size, and Reshaping** | reshape(), resize(), flatten(), ravel(), Transpose, Dimension manipulation |
| 04. | **Array Concatenation & Splitting** | concatenate(), vstack(), hstack(), column_stack(), split(), hsplit(), vsplit() |
| 05. | **Trigonometric Functions** | sin(), cos(), tan(), inverse trig, radians/degrees, Hyperbolic functions |
| 06. | **Random Number Generation** | rand(), randn(), randint(), choice(), distributions, Random seeds |
| 07. | **Array Indexing & Slicing** | Basic indexing, Multi-dimensional slicing, Negative indexing, Step slicing |
| 08. | **Mathematical Operations** | Element-wise operations, ufuncs, sum(), mean(), std(), dot(), matmul(), Statistics |
| 09. | **String Operations** | char.add(), multiply(), upper(), lower(), strip(), split(), Comparison |
| 10. | **Array Broadcasting** | Broadcasting rules, Compatible dimensions, Practical examples, Performance |
| 11. | **Boolean Masking** | Conditional filtering, Boolean indexing, Logical operators, np.where() |
